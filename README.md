# COVID-19-Global-Project

### Overview:
This project analyzes global COVID-19 data (country_wise_latest.csv) to summarize confirmed, active, recovered, and death cases across countries and WHO regions using clean, visual, and meaningful charts.

#### Key Features:
	•	Data cleaning and preparation.
	•	Global and regional summaries.
	•	Top countries by confirmed, death, and recovery rates.
	•	Visualizations: bar charts, pie charts, donut charts.

#### Insights:
	•	Top countries: US, Brazil, India.
	•	Highest death rates: Yemen, UK, Belgium.
	•	Full recovery countries: Dominica, Grenada, Holy See.
	•	Regional trends: The United States had the largest share of confirmed and active cases.
	•	Global totals:
    Confirmed: 16.4M, Recovered: 9.4M, Deaths: 654K, Active: 6.3M

#### Libraries Used:
    pandas, numpy, matplotlib, seaborn.

#### Project Structure:
    1. Complete Kaggle Dataset - This contains the original dataset, which has been downloaded from Kaggle. This is a combination of 4 - 5 datasets. I chose to work on the country_wise_latest.csv dataset.
    
    2. Covid-19 Project - This folder contains my work.
    A cleaned csv file named country_wise_latest_cleaned.csv and a Jupyter notebook named Covid19project.ipynb contains all of my work.
